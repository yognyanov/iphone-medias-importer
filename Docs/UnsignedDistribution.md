# Unsigned Distribution Notes

Bu paket notarized veya Developer ID ile imzalanmis degildir. Bu nedenle baska bir Mac'te ilk acilista Gatekeeper uyarisi gormek normaldir.

Kullaniciya gonderebileceginiz kisa not:

1. `iPhoneMediaImporter.app` uygulamasini `Applications` klasorune surukleyin.
2. Uygulamaya normal cift tiklamak yerine ilk kez `sag tik > Open` yapin.
3. macOS engellerse `System Settings > Privacy & Security` altinda `Open Anyway` secenegini kullanin.

Bu paket test amaclidir. Genel son kullanici dagitimi icin `Developer ID Application` ve notarization gerekir.
